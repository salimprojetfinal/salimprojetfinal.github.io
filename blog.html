Que faire Si votre ordinateur est lent? :

Mon ordinateur est lent, que puis-je faire?.

    Il y a plisieurs cause possible pour qu’un ordinateur devienne trop lent :
  1. Vous utilisez des programmes trop lourds pour être supportés par votre ordinateur
  2. Mémoire vive insuffisante
  3. Disque dur défectueux
  4. Les mises à jour de Windows deviennent de plus en plus lourde pour le système
  5. Il est possible que votre ordinateur soit infecté par des malwares ou de spywares.
    Que faire?
  1. Avant d’acheter un ordinateur, vérifiez toujours ce que ça prend pour faire fonctionner vos programmes si c’est trop tard, vérifier s’il n’y a pas de programmes similaires qui prennent moins de ressources.
  2. Rajouter de la mémoire vive, si vous le pouvez.
  3. Vérifiez l’intégrité de votre disque dur avec le logitiel gratuit hard disk sentinel.
  4. Vous pouvez changer le disque dur par un SSD.
  5. Désinfecter votre ordinateur avec ces programmes gratuits :
  • Rogue killer pour la désinfection des rogues
  • Adw cleaner pour la désinfection des adwares
  • Malwarebytes pour la désinfection programmes indésirables (malware)
  • Glary utility qui est un nettoyeur et réparateur de Windows
